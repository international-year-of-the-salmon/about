Open Access to IYS Data

The International Year of the Salmon Data Mobilization Portal centralizes access to data produced by research expeditions into the high seas of the north Pacific Ocean in the winters of 2019, 2020 and 2022. This portal holds summaries of data publishing as well as technical documentation and policies for project participants.

For public data access, see the IYS Data Catalogue.

How to Provide Data

What is required by IYS scientists providing data?

1) Register the dataset you have collected or will generate by filling out this metadata intake form.

2) Submit data to the IYS (secretariat@npafc.org and cc iys.data@hakai.org) for publishing to the IYS Data Catalogue according to the Data Submission Guidelines. Or create a github repository in the IYS GitHub organization and include the link to your repository in your metadata submission. To be added to the IYS GitHub organization, please first contact the NPAFC secretariat at secretariat@npafc.org.

Remember to cite datasets used in publications according to the recommended citation for each specific dataset in the IYS Data Catalogue
